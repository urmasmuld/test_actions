# PlantTracker_Team2

This is a simple plant tracker built with Vue and Express. It uses plain json arrays to store the data.


## Project management

https://www.pivotaltracker.com/n/projects/2674648

## Project setup

```
cp .env.example .env
npm install
```

## Running the app

```
npm start
```
